# PathBridge 🚀

**Nigeria’s Digital Learning & Career Infrastructure**

PathBridge is a scalable, AI-powered platform designed specifically for Nigerian students to bridge the gap between academic learning and career success. It provides personalized roadmaps, localized study assistance, and direct access to career opportunities.

---

## ✨ Key Features

### 🛠️ Career Path Simulator (Hackathon-Level)
*   **AI-Generated Roadmaps:** Enter any career goal (e.g., "Data Scientist in Abuja") and get a structured, milestone-based path.
*   **Progress Tracking:** Mark milestones as complete and watch your progress bar grow.
*   **Start Learning Tips:** Get instant, actionable AI tips for your current task, tailored to the Nigerian market.
*   **Persistence:** Your progress and roadmap are saved locally, ensuring you never lose your spot.

### 📚 Study & Reading Assistant
*   **Step-by-Step Explainer:** Complex topics broken down with relatable Nigerian examples.
*   **Reading Assistant:** Paste long articles or textbook passages to get simplified summaries, key terms, and takeaways.

### 🎮 Gamified Experience
*   **Missions:** Complete tailored tasks to earn XP and level up.
*   **Rewards:** Interactive feedback and XP system to keep students motivated.

### 💼 Opportunities Dashboard
*   Access localized career opportunities, scholarships, and internships relevant to your chosen path.

---

## 🛠️ Tech Stack

*   **Frontend:** React 19, Vite, Tailwind CSS v4, Framer Motion (Animations).
*   **Icons:** Lucide React.
*   **AI Engine:** Google Gemini API (@google/genai).
*   **Backend:** Express.js (Full-stack architecture).
*   **Language:** TypeScript.

## 🚀 Getting Started for New Users

Follow these steps to get PathBridge running on your local machine.

### 1. Prerequisites
*   [Node.js](https://nodejs.org/) (v18 or higher recommended)
*   [NPM](https://www.npmjs.com/) (usually comes with Node.js)

### 2. Installation
Clone the repository or download the ZIP file, then navigate to the project folder in your terminal:

```bash
# Navigate to the project directory
cd react-example

# Install dependencies
npm install
```

### 3. Environment Configuration
PathBridge requires a Gemini API key to power its AI features.

1.  Create a `.env` file in the root directory (you can copy `.env.example`).
2.  Add your Gemini API key:
    ```env
    GEMINI_API_KEY=your_api_key_here
    ```
    *Note: If no API key is provided, the app will function in "Mock Mode" with simulated responses.*

### 4. Running the App
Start the development server (Full-stack mode):

```bash
npm run dev
```

The app will be available at: **[http://localhost:3000](http://localhost:3000)**
